# 📱 Додавање Икона на TaskFlow Pro за Android ## Проблемот: Ти треба **вистински слики** (PNG фајлови) за иконите. Во `manifest.json` се референцирани `icons/icon-192.png` и `icons/icon-512.png`, но тие фајлови **не постојат** во твојот GitHub repo. --- ## Чекор по чекор решение: ### Чекор 1: Креирај ги иконите Додади го овој фајл `generate-icons.html` - отвори го во browser за да ги генерираш иконите: ```html TaskFlow Icon Generator

🎨 TaskFlow Pro - Icon Generator

Генерирај ги иконите за PWA инсталација

512 x 512

192 x 192

180 x 180 (Apple)

32 x 32 (Favicon)

💡 По преземањето, стави ги фајловите во фолдерот icons/ во твојот GitHub repo, потоа push-ни на GitHub и Vercel автоматски ќе ги deploy-не.

🖼️ Или користи своја слика:


``` ### Чекор 2: Структура на фолдери во GitHub ``` твојот-repo/ ├── index.html ← главната апликација ├── manifest.json ├── sw.js ├── favicon.ico ← (опционално) ├── generate-icons.html ← помошна алатка (може да ја избришеш потоа) └── icons/ ├── icon-192.png ← ЗАДОЛЖИТЕЛНО ├── icon-512.png ← ЗАДОЛЖИТЕЛНО ├── apple-touch-icon.png └── favicon-32x32.png ``` ### Чекор 3: Ажурирај `manifest.json` ```json { "name": "TaskFlow Pro", "short_name": "TaskFlow", "description": "Модерна апликација за задачи и потсетници", "start_url": "./index.html", "scope": "./", "display": "standalone", "orientation": "portrait-primary", "background_color": "#0F0E17", "theme_color": "#6C63FF", "lang": "mk", "categories": ["productivity", "utilities"], "icons": [ { "src": "./icons/icon-192.png", "sizes": "192x192", "type": "image/png", "purpose": "any" }, { "src": "./icons/icon-192.png", "sizes": "192x192", "type": "image/png", "purpose": "maskable" }, { "src": "./icons/icon-512.png", "sizes": "512x512", "type": "image/png", "purpose": "any" }, { "src": "./icons/icon-512.png", "sizes": "512x512", "type": "image/png", "purpose": "maskable" } ], "screenshots": [ { "src": "./icons/icon-512.png", "sizes": "512x512", "type": "image/png", "form_factor": "narrow", "label": "TaskFlow Pro Dashboard" } ] } ``` ### Чекор 4: Додади мета тагови во `index.html` (во ``) ```html TaskFlow Pro ``` ### Чекор 5: Ажурирај `sw.js` ```javascript const CACHE_NAME = 'taskflow-v3'; const ASSETS = [ './', './index.html', './manifest.json', './icons/icon-192.png', './icons/icon-512.png', './icons/apple-touch-icon.png', 'https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;500;600;700;800&display=swap', 'https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css' ]; self.addEventListener('install', event => { event.waitUntil( caches.open(CACHE_NAME) .then(cache => { return cache.addAll(ASSETS).catch(err => { console.log('Cache addAll error:', err); // Try adding one by one return Promise.allSettled( ASSETS.map(url => cache.add(url).catch(() => {})) ); }); }) .then(() => self.skipWaiting()) ); }); self.addEventListener('activate', event => { event.waitUntil( caches.keys() .then(keys => Promise.all( keys.filter(k => k !== CACHE_NAME).map(k => caches.delete(k)) )) .then(() => self.clients.claim()) ); }); self.addEventListener('fetch', event => { // Skip non-GET requests if (event.request.method !== 'GET') return; event.respondWith( fetch(event.request) .then(response => { if (response && response.status === 200) { const clone = response.clone(); caches.open(CACHE_NAME) .then(cache => cache.put(event.request, clone)); } return response; }) .catch(() => caches.match(event.request)) ); }); self.addEventListener('notificationclick', event => { event.notification.close(); event.waitUntil( clients.matchAll({type: 'window'}).then(list => { if (list.length > 0) return list[0].focus(); return clients.openWindow('./'); }) ); }); ``` ### Чекор 6: Push на GitHub → Vercel ```bash # Во терминал, во фолдерот на проектот # 1. Прво генерирај ги иконите # Отвори generate-icons.html во Chrome # Кликни "Преземи ги СИТЕ" # Стави ги во icons/ фолдерот # 2. Провери дали се тука ls icons/ # Треба да видиш: icon-192.png icon-512.png apple-touch-icon.png favicon-32x32.png # 3. Додади ги во git git add . git commit -m "Add PWA icons and update manifest" git push origin main # 4. Vercel автоматски ќе го redeploy-не! ``` --- ## 🔍 Проверка дали работи: ### На телефон (Android): ``` 1. Отвори ја Vercel URL во Chrome 2. Почекај 3-5 секунди 3. Ќе се појави банер "Add to Home Screen" или "Инсталирај" 4. Или: Chrome Menu (⋮) → "Install app" / "Add to Home screen" 5. Иконата ќе се појави на почетниот екран! ``` ### Debugging во Chrome Desktop: ``` 1. Отвори ја страната 2. F12 → Application tab 3. Кликни "Manifest" - провери дали иконите се вчитани 4. Кликни "Service Workers" - провери дали е активен 5. Кликни "Icons" - треба да ги видиш сликите ``` ## ⚠️ Чести проблеми: | Проблем | Решение | |---------|---------| | Иконата не се покажува | Провери го патот `./icons/` наспроти `/icons/` | | "Site cannot be installed" | Мора HTTPS (Vercel е OK) + валиден manifest | | Стара икона се кешира | Chrome Settings → Apps → TaskFlow → Clear data | | manifest не се чита | Провери JSON syntax, нема trailing comma | | Не се појавува Install банер | Потребни се 2+ посети со 5мин размак |
